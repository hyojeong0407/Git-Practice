Git 뉴비임다..ㅎ

git init - 내 PC에서 관리하는 git 저장소

git clone https://git - clone 명령어로 기존 원격 저장소를 로컬로 받을 수 있음

git status - 파일의 상태를 확인하는 명령어

git add - 작업 폴더에서 작업한 변경을 stage에 올릴 때 사용하는 명령어

git commit -m "커밋 메시지" - 스테이지에 추가한 수정 파일을 git 저장소에 저장

git push 원격 저장소 이름 브랜치 이름 - commit한 파일을 원격 저장소에 올리는 명령어

git pull - 원격 저장소의 데이터를 가져오고, 자동으로 현재 작업하는 로컬 브랜치
(pull은 수시로 해주기! - 충돌 현상 생기지 않기 위해)
